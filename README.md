# Rafi's Calendar

A minimal pixel-art desktop calendar widget built with Electron. It sits on your desktop as a small, frameless, transparent window showing the current day and month in a retro pixel aesthetic.

## Download

[Download for macOS (Apple Silicon)](https://github.com/Tanvir-Rafi03/Calendar/releases/download/v1.0.0/Calendar-1.0.0-arm64.dmg)

## Features

- Displays the current date and month in real time
- Frameless and fully transparent window — blends into your desktop
- Pixel-art UI with a custom Pixeboy font and illustrated frame assets
- Draggable window, no taskbar presence (dock icon hidden on macOS)
- Lightweight — no backend, no internet connection required
